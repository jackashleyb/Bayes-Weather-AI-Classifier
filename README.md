# Bayes-Weather-AI-Classifier
A simple Ai style probability calculator that uses Bayes' Theorem and conditional probability to predict weather conditions, this helps to reinforce learning ideas based around my Artificial Intelligence university module 


## Project Aim
the aim of this proejct is to help me understand Bayes' theorem and conditional probabilty in a working python program. 

the program uses evidence such as: 
- Cold Weather
- Rainy Weather

  - Conditional Probability
  - Bayes' Theorem
  - Prior Probability
  - likelihood
  - Evidence
  - Ai decision making under uncertanity
